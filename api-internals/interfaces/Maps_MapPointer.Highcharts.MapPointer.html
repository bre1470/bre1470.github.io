Interface MapPointer

The mouse and touch tracker object. Each Chart item has one associated Pointer item that can be accessed from the Chart.pointer property.

Name

Highcharts.Pointer

Param

The chart instance.

Param

The root options object. The pointer uses options from the chart and tooltip structures.
